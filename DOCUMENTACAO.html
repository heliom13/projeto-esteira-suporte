7. Como Rodar Localmente

Pré-requisitos

  • Java 17 instalado
  • Node.js instalado

Backend

Abra um terminal na pasta metro-api-main e execute:

.\gradlew.bat bootRun

O backend sobe em http://localhost:8080 usando banco H2 em memória (sem precisar de PostgreSQL instalado).

Frontend

Abra um terminal na pasta metro-front-main e execute:

npm install --legacy-peer-deps (apenas na primeira vez)

npm start

O frontend abre automaticamente em http://localhost:3000.

Atenção: Localmente o sistema usa banco H2 em memória. Os dados são perdidos ao reiniciar o backend. O usuário admin@metro.com / admin123 e helioncorrea13@gmail.com / JESUSSALVA são criados automaticamente ao iniciar.

9. Como Fazer Deploy de Alterações

  1. Faça a alteração no código localmente
  2. Abra o terminal na pasta Projeto Esteira Suporte
  3. Execute: git add .
  4. Execute: git commit -m "descrição da alteração"
  5. Execute: git push origin master
  6. O Render detecta automaticamente e faz o deploy (aguarde 5-10 min para o backend)
